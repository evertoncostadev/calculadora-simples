# 🧮 Calculadora Simples

## Propósito
Projeto de um software web para realizar as quatro operações matemáticas básicas (soma, subtração, multiplicação e divisão), executado inteiramente no navegador (frontend) sem dependência de servidor (backend).

## Como Executar
Para testar a calculadora, basta fazer o download dos arquivos e abrir o index.html em qualquer navegador web.
